ОрЮИ МВД России им. В. В. Лукянова

Оснащение актового зала проекционной системой.
Для решения задачи по видеоотображению в актовом зале был использован моторизованный экран шириной 3.5 м соотношение сторон 4:3 и инсталляционный проектор Panasonicс короткофокусным объективом.
Оснащение зала ученого совета.
Оборудование установленное в зале состоит из нескольких подсистем: озвучивание с подавителем обратной акустической связи, что позволяет разместить акустические систему напротив микрофонов, аудиоконференц система Creator включает в себя 16 пультов делегата и 1 пульт председателя, который имеет приоритет для выступления перед другими, система отображения информации две плазменные панели установленные на стены, проектор с экраном, на столе для каждого участника установлен ноутбук для демонстрации презентаций и видео материалов. Для удаленной видеосвязи используется кодек видеоконференции.
Интерактивный лазерный тир «Рубин»
В 2013 году в Орловском Юридическом институте был установлен интерактивный лазерный тир «Рубин». Комплекс состоит из трех проекторов, изображение с которых объединяется в одну панораму. В состав системы входит большое количество прикладных программ. Тренировочные стрельбы трое обучаемых с одновременной регистрацией, обработкой и отображением персональных результатов по каждому стрелку. Тренировка ведется из учебного оружия, оснащенного лазерными тренажерами с невидимым лазерным излучением. Стрельба из АК, РПК одиночными выстрелами и очередями в зависимости от условий упражнения. Использование беспроводных технологий позволяет проводить стрельбу из различных положений с места и в движении.
